The Cube House was the final project of the first year foundational studio at Carnegie Mellon University. The project begain with three planes that were placed into a 20'x20' cube. This house was then placed within an 80'x80' site alongisde Lake Eerie. This house is meant to function as a wedding pavillion and the site is meant to accomodate the wedding ceremony and the reception for 30 guests.

The Drawiwng Performance assignment was the first project of foundational studio part II at Carnegie Mellon University. This project was in collaboration with Carmen Yu. First, each student was given a kitchen tool and asked to draw sections of the tool adn later the motion of the tool. Once collaboration began, each partner combined the motion drawings into a model.


Hunt Library Parasite

This Hunt Library Parasite is the second project in foundational studio II at Carnegie Mellon University. This project is meant to transform parts of the buiding to accomodate study spaces for five-ten people. These two proposals both incorporate a pod attached to the building that interact with lighting, nature, and private space.
